Code Study Method
Use this method for every code file.
Step-by-step method
Run the original code exactly as written.
Record whether it runs successfully.
Identify the purpose of the code.
Explain each import line.
Explain each circuit-construction line.
Explain each execution line.
Explain each output line.
Modify one line only.
Predict the outcome before running.
Run the modified code.
Compare prediction and output.
Record what was learned.
Good modification questions
What happens if I remove this gate?
What happens if I change `h(0)` to `x(0)`?
What happens if I change the number of shots?
What happens if I change the observable from `ZZ` to `XX`?
What happens if I change ansatz parameters?
Does the modified code still make physical sense?
